# wgpu 是什么？

wgpuopen in new window 是基于 WebGPU API 规范open in new window 的、跨平台的、安全的、纯 Rust 图形 API。它是 Firefox、Servo 和 Deno 中 WebGPU 整合的核心。

wgpu 可以在 macOS / iOS、Android、Window 和 Linux 等系统上原生运行，也可以在 Web 环境运行。

WebGPU 是由 W3C GPU for the Webopen in new window 社区组所发布的规范，目标是允许网页代码以高性能且安全可靠的方式访问 GPU 功能。它通过借鉴 Vulkan API，并将其转换为宿主硬件上使用的各式 API（如 DirectX、Metal、Vulkan）来实现这一目标。

WebGPU 规范及 wgpu 都仍在完善阶段，本教程中的部分内容可能会发生变化。如果有错过一些重要的细节，或者解释得不够好，欢迎提出建设性的反馈。

# 为什么使用 Rust？

wgpu 实际上提供了 C 语言绑定 (wgpu-nativeopen in new window)，你可以写 C/C++ 或其他能与 C 互通的语言来使用它。尽管如此，wgpu 本身是用 Rust 实现的，它便利的 Rust 绑定能减少你使用中的阻碍。更重要的是，Rust 是一门高性能，内存和线程安全且极具生产力的现代底层语言。

在学习本教程之前你需要先熟悉 Rust，因为这里不会详细介绍 Rust 的语法知识。如果对 Rust 还不太熟悉，可以回顾一下 Rust 教程open in new window或 Rust 语言圣经open in new window。另外还需要熟悉 Rust 包管理工具 Cargoopen in new window。

如果你还没有学习过任何一门底层语言，那就学 Rust 吧？它不像 C++ 有那么多复杂难懂的概念;

如果你觉得学 C/C++ 很难，那就学 Rust 吧，你会受益于它的高性能，内存和线程安全;

如果你已精通 C/C++，那也学学 Rust 吧，那么难的都学得会，掌握 Rust 不过是小菜一碟;
